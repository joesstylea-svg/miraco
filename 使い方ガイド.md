# MIRACO 使い方ガイド

スマホで完結する「受注・やり取り・お支払い・納品」アプリ。
ユーワード活動の小さな「困った」を、申込みからやり取りまでワンストップで受け付けます。
※ メニュー名・色・呼び方を変えれば、占い・教室・マッサージなど他業種にもそのまま使えます。

---

## 🧑‍🤝‍🧑 お客様の使い方

### 1. 申し込む
1. ホーム or メニューから、気になるメニューを選ぶ
2. 「このメニューを申し込む」→ ウィザードで入力
   - サービス内容の選択（デザイン・フォントなど／画像付きで選べる）
   - お名前・メール・連絡方法（LINE / アプリ内チャット）
   - ご連絡事項・参考画像・希望納期・お支払い方法
3. 金額（基本料金＋オプション）が自動計算されて表示
4. 確認 → 送信 → 受付完了

### 2. マイページで確認
- 申込み後はそのままログイン状態に
- カードで「受注日・納期目安・ステータス・支払い状況」が一目でわかる
- **詳細を見る**：申込み内容・見積り・進捗ステップ・納品データ
- **チャットメッセージ / LINEメッセージ**：オーナーとやり取り（画像も送れる）
- **お支払い**：金額を見て、案内された方法（PayPay・振込・リンク等）で支払う

### 3. 納品・レビュー
- 納品データはマイページから受け取り（共有リンク or 画像DL）
- 納品後は★レビューを投稿（掲載は運営が選択）

### 別端末でログイン
- マイページでメールアドレスを入力 →「ログイン」

## 🛠 運営者（オーナー）の使い方

### 管理画面に入る
- 右上メニュー（≡）→ 一番下の **⚙運営者** アイコン → パスワード入力
- 初期パスワード：`miraco`（管理設定で変更可）

### 日々の流れ
1. **新着**の依頼が入る（バッジで通知）
2. 依頼カードをタップ → 詳細モーダル
   - チャットで質問・確認（定型文ワンタップあり）
   - 担当者・社内メモ（お客様非表示）
   - 入金確認できたら「支払い済みにする」→ 自動で「制作中」へ
   - 納品データを追加（Googleドライブ等の共有リンク推奨）→ 自動で「確認中」へ
   - お客様の確認OKが来たら「納品済み」に変更
3. フィルター（新着 / 対応中 / 未払い / 支払い済み / 納品済み）で絞り込み
4. 統計カード（新着・対応中・売上）タップでも絞り込み
5. CSV書き出しで売上集計・確定申告にも

### サイトを育てる（管理設定）
- **ホーム画面を編集**：ロゴ・ブランド名・見出し・「困った」セクション・並び替え（ブロック式）
- **ギャラリーを編集**：制作例・お客様の声をLP式に
- **お支払い方法を編集**：方法名・案内文・支払い先情報・決済リンク
- **メニュー管理**：追加・編集（料金・納期・LP・申込フォーム・公開停止・バッジ）
- **レビュー管理**：投稿されたレビューの公開/非公開を選択
- **カラーテーマ**：5パターンから選択
- **業種ワード**：「メニュー→講座」「申し込み→予約」などに言い換え
- 編集中は **👁プレビュー** で保存前に確認

## ⚙ 仕組み・運用メモ

- **データ保存**：Supabase（クラウド）に保存。複数端末で共有・自動同期（約12秒ごと）
- **オフライン**：IndexedDBにキャッシュ
- **公開**：GitHub Pages（無料）。独自ドメインも後付け可
- **連絡方法**：管理画面で LINE / アプリ内チャット / 両方 を切替（既存注文は申込時のまま）
- **納品**：共有リンク方式なら容量を気にせず大きいファイルもOK・無料

### 今後の拡張候補
- 本物のメール認証（Supabase Auth マジックリンク）※一般公開時に
- 注文を1件1行のテーブルに分割（同時アクセスに強くする）
- リアルタイム通信の高速化（現在はポーリング）

## 📣 紹介コピー（画像素材などに）

> **「何から始めたらいい？」を、申込みからやり取りまでスマホひとつで。**
> 名刺・アイキャッチ・出店準備…小さな「困った」を、やさしく受け止めます。
> 申込み → 見積り自動計算 → チャットで相談 → お支払い → 納品まで、ぜんぶこの中で完結。
